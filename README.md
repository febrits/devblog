# DevBlog — Full-Featured Blog Platform

A modern blog platform with search, tags, and about page.

**Live:** https://devblog-three-ochre.vercel.app

## Features
- Create, edit, and delete blog posts
- Search and filter by tags
- About page
- Responsive dark theme

## Tech Stack
React 19 · TypeScript · Tailwind CSS v4 · Vite · React Router

## Getting Started
```bash
npm install
npm run dev
```
